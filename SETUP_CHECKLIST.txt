== AVANT DE COMMENCER ==
✅ Créer un compte GitHub (https://github.com)
✅ Créer un nouveau repo
✅ Avoir Git installé sur Windows

== ÉTAPES ==

1. CRÉER LE REPO GITHUB
   - Va sur https://github.com/new
   - Crée un repo (ex: EdajcFlutterApp)
   - NE copie pas de fichier - repo vide

2. EXÉCUTER LE SCRIPT
   - Double-click sur push_to_github.bat
   - Entre l'URL de ton repo
   - Attends que ça se termine

3. ATTENDRE LE BUILD
   - Va sur GitHub > Actions
   - Attends 10-15 minutes
   - Télécharge l'APK quand c'est ✅

4. PROCHAINES FOIS
   - git add .
   - git commit -m "description"
   - git push origin main
   - L'APK se buildera automatiquement
